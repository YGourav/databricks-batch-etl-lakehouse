# Databricks Batch ETL Lakehouse Project

## Overview
This project demonstrates an end-to-end batch data engineering pipeline built using Databricks and Apache Spark SQL.  
The pipeline follows the Lakehouse architecture (Bronze, Silver, Gold) to transform raw data into analytics-ready datasets.

## Architecture
- **Bronze Layer**: Raw data ingestion
- **Silver Layer**: Data cleaning and transformations
- **Gold Layer**: Aggregated business-level tables

## Tech Stack
- Databricks
- Apache Spark / Spark SQL
- Delta Lake
- SQL
